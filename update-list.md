# Update List

아래 항목은 추후 작업 예정인 업데이트 목록입니다.

1. 이미지 압축 기능 추가
   - Base64 기술을 사용해 이미지 압축 기능을 구현한다.

2. 동영상 파일 선택/드래그앤드롭 필터링 강화
   - 파일 선택 창에서 video 파일만 보이도록 제한한다.
   - 드래그 앤 드롭 시 비디오 형식이 아닌 파일은 자동으로 리스트에 표시되지 않도록 처리한다.

3. 라이센스 등록 후 확인 화면 및 PC 이동 기능
   - 라이센스 등록 완료 후 "내 라이센스 체크" 화면을 표시한다.
   - 해당 화면에 PC 이동 관련 버튼(기기 이전/이동 기능)을 추가한다.

4. 동영상 압축 옵션에 AI 추천 모드 추가
   - "AI 추천 모드" 옵션을 추가한다.
   - 실제 AI 추천이 아닌, MPEG 압축이 가능하도록 구성한다.
